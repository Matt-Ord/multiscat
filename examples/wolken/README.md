## Wolken HeLi Scattering from a Corrugated Surface

This example demonstrates how to use multiscat to simulate scattering from the
Wolken HeLi surface.

This model was originally developed by [Wolken (1973)](https://doi.org/10.1063/1.1679617).
It was used in [Manolopoulos et al. (1990)](https://doi.org/10.1039/FT9908601641) to compare
the performance of the Log-Derivative technique, which we use here as a comparison.
